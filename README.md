Xylophone App

This is a xylophone app which is developed using flutter.

Steps to view the xylophone:

Step 1: Download the zip code of this xylophone app.

Step 2: Extract the zip file.

Step 3: Open the folder in the Visual Studio Code.

Step 4: Open the terminal ans run the command : flutter run -d chrome --release

You will be able to view the xylophone app your chrome browser.
